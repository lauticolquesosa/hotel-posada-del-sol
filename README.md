# Hotel Posada del Sol — Salta

Sitio web (landing) del Hotel Posada del Sol, en Alvarado 646, centro histórico de Salta, Argentina.

Rediseño estático (HTML + CSS + JS vanilla), bilingüe ES/EN, con reservas directas por WhatsApp.

## Estructura
- `index.html` — página completa
- `app.js` — lógica (idioma, menú, reservas, animaciones de scroll, lightbox)
- `assets/` — imágenes y logos

## Desarrollo local
```bash
python -m http.server 8000
# abrir http://localhost:8000
```
